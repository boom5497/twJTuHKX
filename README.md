# 前言

欢迎来到基于SSM的在线日语培训系统项目！本项目旨在为广大日语学习爱好者提供一个便捷、高效的在线学习平台。通过使用Java语言和Spring、SpringMVC、MyBatis框架，结合前端技术如JS、Vue和CSS3，我们致力于打造一个功能完善、用户体验优良的在线日语培训系统。

# 内容介绍

本项目主要包括以下功能模块：课程展示、视频教学、作业提交与批改、在线测试、讨论区等。用户可以根据自己的需求和进度进行学习，实现个性化教学。此外，系统还为教师提供了便捷的课程管理和学生管理功能，提高教学效率。

# 技术介绍

- 语言：Java
- 使用框架：Spring、SpringMVC、MyBatis
- 前端技术：JS、Vue、CSS3
- 开发工具：IDEA/Eclipse
- 数据库：MySQL 5.7/8.0
- 数据库管理工具：phpstudy/Navicat
- JDK版本：jdk1.8
- Maven：apache-maven 3.8.1-bin
- 前端环境：Node.Js 12\14\16
